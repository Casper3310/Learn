Laravel Homestead開發環境設定

參考影片 參考影片 在Homestead中安裝phpMyadmin 安裝中文手冊

需安裝

  • VirtualBox
  • Vagrant
  • phpMyadim

    • 複製完 Homestead 儲存庫，即可在 Homestead 目錄中執行 bash init.sh 指令來建立 Homestead.yaml 設定檔。 Homestead.yaml 檔案將會被放置在 ~/.homestead 隱藏目錄中：

    // Mac / Linux... bash init.sh

    // Windows... init.bat

    Windows可直接執行init.bat批次檔產生Homestead.yaml設定檔

    Homestead.yaml設定檔中

    folders:
    本機位置- map: ~/Code
    對應虛擬機位置to: /home/vagrant/Code

    sites:
    本機網址- map: homestead.test
    對應虛擬機位置to: /home/vagrant/TRTC_Homestead/TRTC/public

    在C:\Windows\System32\drivers\etc\hosts檔案中可以設定本機網址與虛擬機對應關西

    預設虛擬機IP 192.168.10.10 homestead.test加入檔案中

    vagrant 指令

    vagrant init初始化
    vagrant up啟動
    vagrant halt關閉
    vagrant status機器開關狀態
    vagrant ssh用SSH連線虛擬機
    vagrant destroy刪除虛擬機
    vagrant destroy --f強制刪除虛擬機
    vagrant box list現在box清單
    vagrant provision更新

    linux指令

    pwd顯示目前所在的目錄
    ls清單
